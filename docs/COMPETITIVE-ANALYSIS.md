# Impactt Gym — Competitive Analysis

**Purpose:** Pitch document for gym owners evaluating Impackt vs existing tools.

---

## Market Context (India, 2025–2026)

Indian gym software (Akton, Kasratbook, GymKloud, Gympe) focuses on **membership management, attendance, and billing**. Global tools (Mindbody, Trainerize, TrueCoach) target broader or trainer-only markets.

Impackt occupies a **narrow, high-value niche:** gyms where personal training revenue is significant and split calculations + trainer payroll are painful.

## What Impackt Solves Better

### 1. PT Revenue Split Transparency
**Problem:** Owners manually calculate "Trainer A gets 40%, Trainer B gets 55%" each month.  
**Impackt:** Split % set once per trainer; every payment auto-computes shares. Owner dashboard shows live MTD totals.

### 2. Salary + Commission in One Flow
**Problem:** Indian gym owners use one spreadsheet for PT and another for payroll.  
**Impackt:** Salaries module pulls PT commission automatically, adds base salary + incentives, generates PDF pay stub.

### 3. Trainer Day Planner
**Problem:** Trainers don't know which slots are free for new trial clients.  
**Impackt:** Visual slot management — open, blocked, or booked — designed for in-gym mobile use.

### 4. Renewal Pipeline
**Problem:** Renewals chased on WhatsApp; revenue leaks when packages expire silently.  
**Impackt:** Owner and trainer see clients due in 7/30 days on dashboard.

### 5. Supervisor Without Full Admin
**Problem:** Managers need to run payroll but shouldn't change business rules.  
**Impackt:** Dedicated Supervisor role — operations + record payments, no split/salary rule edits.
